# RACeD: Reference Assurance Case Guidance Demonstrator

RACeD is a prototype guidance environment for applying GSN-based assurance
patterns developed using the AIR reference case as the running example.

The demonstrator presents pattern guidance, expected evidence prompts,
limitations, and selected GSN visualisations for entered parameter bindings.

It does not inspect evidence, run verification tools, check compliance, or
approve a safety claim.
